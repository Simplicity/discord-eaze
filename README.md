### Overview

This repository houses a class-based discord bot structure using discord.js. 

### Installation

You can either install this package via [yarn](#using-yarn) or [npm](#using-npm). 

#### Using yarn

- `yarn add discord-eaze`

#### Using npm

- `npm install discord-eaze`
